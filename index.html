HTML Cheat Sheet

Common HTML Tags

Tag Name Description
<html> HTML The wrapper element for the entire HTML document
<head> HTML Head Sets off the title, css links and font sources, and isn't visible on the webpage
<body> HTML Body Contains the portion visible on the webpage

Table Tags

Tag Name Description
<table> Table The wrapper element for HTML tables
<thead> Table Header The set of <th> elements that define the columns
<th> Header Column Data Data for column titles within <thead> elements
<tbody> Table Body Wraps the body content of tables
<tr> Table Row Table row container
<td> Table Data Cells within a table row
<tfoot> Table Footer Set of rows defining the footer of a table

Table Attributes

Attribute Name Description
colspan Column Span Defines how many columns a <td> element should span
rowspan Row Span Defines how many rows a <td> element should span
